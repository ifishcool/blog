# 欢迎来到 ArenaPro React

## 什么是 ArenaPro React？

**ArenaPro React 是基于 React 18 定制的、适用于神岛的 React 风格 UI 框架。**

- 在 **心智模型与 API 风格上贴近 React 18**，比如组件、Hooks、声明式 UI 等；
- 在 **实现层面与运行环境上做了针对神岛引擎的改造与裁剪**，以便更好地适配神岛的 UI 与性能需求。

### ArenaPro React 相对标准 React 的差异大致包括：

- **运行环境不同**  
  标准 React 主要面向浏览器 DOM 或 Native；  
  ArenaPro React 面向神岛引擎的 UI 系统和渲染管线。

- **API 与能力有取舍**  
  我们保留了大部分常用的 React 18 开发模式，同时对在神岛场景下不常用或不适配的部分做了精简与封装。

- **默认实践更贴近游戏创作场景**  
  在布局方式、组件抽象、性能优化策略等方面，都更加贴合神岛创作的实际项目需求。

## 代码块示例

### JavaScript 代码

```javascript
// React 函数组件示例
const MyComponent = ({ name, count }) => {
  const [localCount, setLocalCount] = useState(count);

  const handleClick = () => {
    setLocalCount(localCount + 1);
  };

  return (
    <div>
      <h1>Hello, {name}!</h1>
      <p>Count: {localCount}</p>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
};
```

## React 组件示例

下面是一个简单的 React 组件示例，展示了如何在 MDX 中嵌入交互式组件：

export const Counter = () => {
  const [count, setCount] = React.useState(0);
  return (
    <div className={'counterContainer'}>
      <h3 className={'counterTitle'}>计数器组件</h3>
      <p className={'counterDisplay'}>
        当前计数2: <strong>{count}</strong>
      </p>
      <div className={'buttonGroup'}>
        <button
          onClick={() => setCount(count + 1)}
          className={`button buttonIncrement`}
        >
          +1
        </button>
        <button
          onClick={() => setCount(count - 1)}
          className={`button buttonDecrement`}
        >
          -1
        </button>
        <button onClick={() => setCount(0)} className={`button buttonReset`}>
          重置
        </button>
      </div>
    </div>
  );
};

## 特性展示

这个示例展示了 MDX 的强大功能：

1. **组件嵌入**: 可以直接在 Markdown 中嵌入 React 组件
2. **状态管理**: 组件可以有自己的状态（如计数器的 count）
3. **事件处理**: 支持点击事件等交互
4. **样式定义**: 可以使用内联样式或 CSS 类

这就是 MDX 相比普通 Markdown 的优势 - 让文档变得生动和交互式！
